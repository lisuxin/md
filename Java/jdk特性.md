# JDK 新特性

## Java SE 5

* 泛型

   ```java
   ArrayList list=new ArrayList();
   ------>ArrayList<Integer>list=new ArrayList<Integer>();
   ```

*  自动装箱/拆箱==原始类型与对应的包装类不用显式转换==

   ```java
   int i=list.get(0).parseInt();
   -------->int i=list.get(0);
   ```

* for-each

   ```java
   for( i=0;i<a.length;i++){......}
   ------------>for(int i:a){......}
   ```

* static import

   ```java
   Math.sqrt();--------------->sqrt();
   ```
